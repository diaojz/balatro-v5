# 小丑牌 Balatro v5

Balatro 风格的扑克牌游戏网页版 — Docs-v5 提示词成熟度验证实现。

## 玩法

3 关盲注（300 / 500 / 800 分）+ 商店 + 6 张 Joker 库。每关 4 手内打到目标分数通关。

## 开发

```bash
npm install
npm run dev     # 启动开发服务器
npm run build   # 构建生产版本
```

## 技术栈

- Vue 3 + Vite + Tailwind CSS
- 无后端、无登录、纯静态
